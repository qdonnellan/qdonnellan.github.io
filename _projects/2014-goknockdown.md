![goKnockdown](/images/goknockdown.png)

One of my favorite features of [lessonwell][1] is a real-time Markdown/MathJax editor that I created to make prototyping math lessons easier. GoKnockdown is an iteration of the same side-by-side editor used in lessonwell, but as a standalone sideproject. I used it as an excuse to learn Go and KnockdownJS on Google App Engine.
